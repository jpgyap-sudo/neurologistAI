# Active Hydrocephalus Red Flags

Track these because they may increase concern for active/progressive hydrocephalus:
- serial ventricular enlargement over time
- transependymal edema/periventricular FLAIR hyperintensity not explained by gliosis
- sulcal effacement/compressed convexity sulci
- DESH pattern
- worsening arousal unexplained by infection, medication, metabolic issue
- headache/vomiting/papilledema if assessable
- rising ICP if measured

If absent and sulci are widened with lesion-side tissue loss, ex-vacuo is more likely.
