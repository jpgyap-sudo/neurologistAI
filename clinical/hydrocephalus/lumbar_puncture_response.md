# Lumbar Puncture Response Module

## Why it matters
Large-volume lumbar puncture response is a clinically important predictor of shunt responsiveness in suspected NPH.

## Dad-specific known fact to encode
- LP performed twice.
- Approximately 40 mL removed each time.
- No observed benefit.

## Interpretation logic
If large-volume LP 30–50 mL x2 shows no objective improvement:
- strongly lowers probability of shunt-responsive NPH
- supports ex-vacuo or non-CSF-pressure explanation
- does not make NPH impossible
- requires stronger imaging evidence before shunt benefit argument is convincing

## Limitation in MCS
Classic NPH LP testing uses gait and cognition. In MCS, gait/cognition are not testable. Use alternative objective markers:
- arousal duration
- visual fixation/pursuit
- command following
- reproducible motor response
- swallow latency
- autonomic stability
- CRS-R score

## Report wording
"Negative LP response x2 reduces expected probability of meaningful shunt benefit, especially without strong imaging signs of active hydrocephalus. Interpretation is limited because gait/cognitive testing is not possible in MCS."
