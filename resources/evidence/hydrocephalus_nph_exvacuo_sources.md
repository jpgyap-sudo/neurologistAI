# Hydrocephalus / NPH / Ex-Vacuo Evidence Scaffold

## Features supporting ex-vacuo ventriculomegaly
- Ventricular enlargement proportional to tissue loss.
- Asymmetric enlargement adjacent to encephalomalacia or chronic infarct/hemorrhage cavity.
- Widened sulci/fissures consistent with atrophy.
- No convincing DESH pattern.
- No progressive transependymal CSF edema pattern.

## Features supporting NPH / communicating hydrocephalus
- Ventriculomegaly out of proportion to cortical sulcal widening.
- Tight high-convexity sulci.
- Disproportionately enlarged Sylvian fissures.
- Reduced callosal angle on correct coronal plane.
- Gait/cognition/urinary triad where assessable.
- Objective response after tap test or external lumbar drainage.

## Measurements to implement
- Evans index
- Callosal angle
- Temporal horn width
- Ventricular asymmetry index
- Lesion-side ventricular expansion index
- Sulcal width proxy
- Serial ventricular volume trend
